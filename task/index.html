Talks on SFD13



Talks on SFD 2013:


These are the topics that we will cover in SFD13

Day 1:

  1. What is SFD?    [ 15 min ]

    1. What is the meaning of  Free Software?

    2. Why it matters?

  2. Introduction to Open Source software and Linux Distributions  [ 15 min ]

    1. Various Distributions like Ubuntu, CentOS, Fedora, Backtrack and their origins

  3. OS Installation Guide (Linux-Mint) [ 1hr ]

    1. What is a live USB and ways to make it? (unetbootin, LiLi and others)

    2. How to boot and configure installation options?

    3. Demo Installation using VirtualBox

    4. Warnings - Common mistakes that usually ruins the installation ! ! !

  4. How to navigate the Linux desktop, using applications and installation. [ 1 hr ]

    1. Virtual desktops, workspace, activities

    2. Demo of different desktop environments like unity, gnome, kde, xfce, lxde.

    3. apt-get usage

    4. How to install desktop environments

  5. General office and internet applications [ 15 min ]

    1. Libreoffice

    2. Chromium

    3. Vim, Emacs (gedit, kate, notepad++)

    4. Games

  6. How to get help and support and communicate. [ ½ hr ]

    1. Mailing Lists

    2. PULUG

    3. IRC

  7. Introduction to Command Line Interface [ 1+ hr ]

    1. Different commands

    2. Why CLI is awesome?

    3. Shell Scripting, permissions etc.

  8. Programming with Linux [ 1+ hr ]

    1. How to program using various languages in linux?

    2. Hello World of C,C++,Java, Python

    3. Programming Contests

    4. Problems faced with online judges of these contests

Day 2:

  1. Introduction to Developing under Linux [ 15 min ]

    1. What is linux?

    2. How linux was developed?

  2. Contributing to Open Source projects [ 1 hr ]

    1. KDE

    2. Gnome

    3. Mozilla

    4. linux-kernel

  3. Introduction to git [ 15 min ]

    1. What is git?

    2. git pull, push, clone

    3. git diff

  4. Using make [ ½ hr ]

    1. What’s make?

    2. What’s inside Makefile?

    3. Build a project during the session

  5. Introduction to development with Qt  [ ½ hr ]

    1. QtCreator Interface

    2. cmake & qmake projects

    3. Build and Run one software in the session

  6. Getting a job in Open Source Development    [ ½ hr ]

    1. How to find a task, junior jobs etc. in a project.

    2. Show them the buglists.

  7. Google Summer of Code and other open source development competitions [ 2 hr ]

    1. Student Talks

    2. Experience and How it happened?

    3. KP Sir: Everything I have learned so far!

    4. Shailja Kumari: freelance editor-translator, Open Technology Institute (OTI), GNOME Outreach Program for Women (OPW) intern

    5. More

  8. Python - A Beginner's Guide

  9. Introduction to Android Development

  10. HTML5 Development Basics



Dates are not final, but they will be after Aavishkaar and Sessionals.

28 and 29th (Sat. & Sun.) maybe!



Date    :     28th Saturday & 29th Sunday

Expected Venue    :    Golden Jubilee Hall (Limited seats ~150)
